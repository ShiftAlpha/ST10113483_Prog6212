#ST10113483 Prog6212

Using this tool, you will be able to track the amount of time you have spent studying for each module over the course of the academic year.
You can view the amount of time you have spent studying for a particular module, as well as a recommendation for how much time you should devote to studying.

Powered by the ASP.net core web application technology, the software grants you access to your data regardless of your location.

Instructions on How to Install
1. Clone repository.
2. Start Visual Studio 2022 and open the project.
The necessary dependencies will be obtained from nuget when the project is constructed.

Directions
Visit the website at https://ST10113483Prog.azurewebsites.net and either sign up for an account or sign in if you already have one.
